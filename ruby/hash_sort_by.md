# Hash#sort_by

Hashをソートする方法

```ruby

hash = {"A"=>{:x=>80,:y=>3}, "B"=>{:x=>80,:y=>2}, "C"=>{:x=>1,:y=>90}}

hash.sort_by{|k, v| v[:x]}
=> [["C", {:x=>1, :y=>90}], ["A", {:x=>80, :y=>3}], ["B", {:x=>80, :y=>2}]]

hash.sort_by{|k, v| v[:y]}
=> [["B", {:x=>80, :y=>2}], ["A", {:x=>80, :y=>3}], ["C", {:x=>1, :y=>90}]]

hash.sort_by{|k, v| [v[:x], v[:y]]}
=> [["C", {:x=>1, :y=>90}], ["B", {:x=>80, :y=>2}], ["A", {:x=>80, :y=>3}]]

```
